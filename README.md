How to start:
1. ```git clone https://github.com/AsetB/DrevmassDemo.git```
2. Install [XcodeGen](https://github.com/yonaskolb/XcodeGen)
3. Generate .xcodeproj file with ```xcodegen generate```

Dependencies:
 - Alamofire
 - KeychainSwift
 - SnapKit
 - SVProgressHUD
 - SwiftyJSON
 - PanModal
 - YouTubePlayerKit
 - SkeletonView
